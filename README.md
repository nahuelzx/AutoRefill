# AutoRefill

![C++](https://img.shields.io/badge/language-C%2B%2B17-blue)
![Platform](https://img.shields.io/badge/platform-Windows-lightgrey)
![Status](https://img.shields.io/badge/status-active-success)

A small **C++ utility for Minecraft Bedrock** that quickly refills your hotbar from the inventory using a single hotkey.

This project was made mainly as a **practice project while learning C++ and Windows input automation**.

---

## Features

* Select **your screen resolution**
* Choose **how many hotbar slots** to refill
* Set **your own hotkey**
* Lightweight and fast

---

## Download

Download the compiled `.exe` from the **Releases** page and run it.
